File manager
============
Noro contains another built in program: the file manager
It can be opened through the menu or by using the files command

Keybinds:
- Up       : Move caret up
- Down     : Move caret down
- Space    : Open directory
- Home     : Go to first file in list
- End      : Go to last file in list
- d        : Delete selected file
- r        : Rename selected file
- o        : Open selected file in the text editor program
- Ctrl + O : Open folder
- Ctrl + R : Reload file list
